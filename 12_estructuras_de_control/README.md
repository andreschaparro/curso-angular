# Capítulo 12: estructuras de control

- `ngIf`: se utiliza para mostrar u ocultar un elemento en base a un valor booleano.
- `ngFor`: se utiliza para iterar sobre una lista y renderizar elementos repetitivos.
- `ngSwitch`: muestra un bloque de contenido diferente en base a un valor.
- `ngClass`: permite cambiar dinámicamente las clases CSS de un elemento.
- `ngStyle`: permite cambiar dinámicamente el estilo de un elemento.
- `ngContainer`: permite agrupar elementos sin agregar `<div></div>` o `<span></span>`.
